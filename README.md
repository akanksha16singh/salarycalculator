# SPJIMR Salary Calculator

A Next.js app to calculate in-hand salary, tax breakdown, domain insights, and 10-year projections based on SPJIMR placement data (Batch 2024–26).

## Features
- 🌗 Light / Dark theme toggle
- 💰 Salary breakdown with deduction details
- 📊 Domain-wise placement insights
- 📈 10-year career growth projection chart
- 🧮 New Tax Regime FY 2025-26 calculator

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Vercel auto-detects Next.js — click **Deploy**

No environment variables needed.
